## Overview

Oslin is an AI-powered video interview platform that enables recruiters to conduct asynchronous video interviews with candidates. The platform provides automated analysis, question management, and a seamless candidate experience.

## Getting Started

### For Interviewers

1. **Sign In**: Access the platform using your credentials
2. **Dashboard**: Navigate to the interviewer dashboard to manage job profiles and interviews
3. **Create Job Profile**: Set up a new job profile with questions and qualifications

### For Candidates

1. **Receive Link**: You'll receive a unique interview link via email
2. **Access Interview**: Click the link to access your interview
3. **Complete Interview**: Follow the on-screen instructions to complete your interview

## For Interviewers

### Creating a Job Profile

1. **Navigate to Dashboard**: Click "Create Job Profile" from the dashboard
2. **Basic Information**:
   - Enter job title
   - Add job description
   - List required qualifications

3. **Interview Questions**:
   - Add questions one by one
   - Set time limits for each question (optional, default: 120 seconds)
   - Enable/disable retakes per question
   - **New Feature**: Add "Please Elaborate" text for additional context
     - Enter explanatory text that appears when candidates click "Please Elaborate"
     - Set extension time (default: 10 seconds) added when elaboration is requested

4. **Question Management**:
   - Drag and drop questions to reorder
   - New questions appear at the top for easy editing
   - Questions save in the order you entered them (Q1 remains Q1)
   - Enable "Shuffle questions" to randomize order for each candidate

5. **Post-Interview FAQ**:
   - Add frequently asked questions that candidates see after completing the interview
   - Each FAQ item includes a question and answer
   - Candidates can review these after submission

6. **Save Profile**: Click "Create Profile" to save

### Editing a Job Profile

1. Click "Edit" on any job profile card
2. Modify any fields as needed
3. Update questions, FAQ items, or settings
4. Click "Update Profile" to save changes

### Managing Interviews

1. **View Candidates**: Navigate to the "Interviews" tab to see all candidate responses
2. **Review Responses**: Click on any interview to view:
   - Candidate information
   - Video responses for each question
   - AI-generated analysis and scores
   - Strengths and weaknesses
   - Communication style assessment

3. **Share Interview Links**:
   - Use the public link to share with candidates
   - Or create unique invite links for specific candidates

### Global Intro Questions

1. **Access Settings**: Go to the Settings tab in the dashboard
2. **Manage Intro Questions**: Add questions that appear at the start of every interview
3. **Features**:
   - Intro questions never shuffle
   - Always appear first in the interview
   - Can include elaborate text like regular questions

## For Candidates

### Starting an Interview

1. **Access the Link**: Click the interview link provided
2. **Enter Information**:
   - Full name
   - Email address
3. **Review Instructions**: Read the interview guidelines
4. **Click "I'm Ready to Start"**

### During the Interview

1. **Question Flow**:
   - Each question appears one at a time
   - Timer shows prominently in the center when question starts
   - Timer moves to top-right after 3 seconds
   - Time remaining is clearly displayed

2. **Recording**:
   - Recording starts automatically
   - Red indicator shows recording status
   - Cannot pause or stop recording

3. **"Please Elaborate" Feature**:
   - If available, a "Please Elaborate" button appears
   - Click to see additional context about the question
   - Automatically extends your time by the configured amount (default: 10 seconds)
   - Can only be used once per question

4. **Moving Forward**:
   - Wait for timer to complete, OR
   - Click "Next Question" after recording for at least 5 seconds
   - Brief intermission between questions (5 seconds)

5. **Progress**:
   - Progress bar at bottom shows completion percentage
   - Question counter shows current position

### After Completing the Interview

1. **Completion Screen**:
   - Confirmation message
   - Your email address for reference

2. **FAQ Section**:
   - Review frequently asked questions
   - Answers to common questions about the position
   - Scrollable list of Q&A items

3. **Next Steps**:
   - The hiring team will review your responses
   - You'll be contacted via email regarding next steps

## Features

### Question Management

- **Drag and Drop**: Reorder questions by dragging
- **Shuffle Option**: Randomize question order for each candidate
- **Time Limits**: Set custom time limits per question
- **Retake Control**: Enable/disable retakes per question
- **Elaborate Text**: Add additional context that candidates can request

### Timer Features

- **Prominent Display**: Timer appears in center when question starts
- **Smooth Animation**: Transitions to top-right after 3 seconds
- **Clear Indication**: Shows "Time left: MM:SS" format
- **Extension Support**: Time extends when "Please Elaborate" is used

### Post-Interview FAQ

- **Flexible List**: Add as many FAQ items as needed
- **Easy Management**: Add, edit, or delete FAQ items
- **Candidate Access**: Automatically shown after interview completion
- **Clear Format**: Numbered questions with detailed answers

### AI Analysis

- **Automated Scoring**: Overall score (0-100)
- **Strengths & Weaknesses**: Identified automatically
- **Communication Style**: Assessment of speaking style
- **Confidence Level**: Evaluation of candidate confidence
- **Skill Alignment**: Match with job requirements
- **Red Flags**: Potential concerns highlighted

## Best Practices

### For Interviewers

1. **Question Design**:
   - Keep questions clear and concise
   - Use "Please Elaborate" for complex questions
   - Set appropriate time limits (60-180 seconds recommended)
   - Test questions before sharing

2. **FAQ Management**:
   - Include 5-10 common questions
   - Cover topics like:
     - Next steps in the process
     - Timeline for decisions
     - Benefits and culture
     - Remote work policies
     - Team structure

3. **Time Limits**:
   - Consider question complexity
   - Allow enough time for thoughtful answers
   - Use elaborate text for questions needing context

4. **Shuffle Questions**:
   - Enable for fairness across candidates
   - Helps prevent answer sharing
   - Maintains interview integrity

### For Candidates

1. **Preparation**:
   - Test your camera and microphone beforehand
   - Ensure good lighting and quiet environment
   - Have a stable internet connection

2. **During Interview**:
   - Speak clearly and at a moderate pace
   - Make eye contact with the camera
   - Use "Please Elaborate" if you need clarification
   - Take advantage of the full time if needed

3. **Technical Tips**:
   - Use Chrome or Firefox for best compatibility
   - Grant camera/microphone permissions when prompted
   - Don't refresh the page during the interview
   - Complete in one session if possible

## Troubleshooting

### Common Issues

**Camera/Microphone Not Working**:
- Check browser permissions
- Ensure no other applications are using the camera
- Try refreshing the page
- Use a different browser

**Timer Not Showing**:
- Wait 3 seconds after question starts
- Check top-right corner of screen
- Timer should show "Time left: MM:SS"

**"Please Elaborate" Button Missing**:
- This feature is optional per question
- Only appears if interviewer added elaborate text
- Not available for all questions

**FAQ Not Showing After Completion**:
- FAQ is optional per job profile
- Only appears if interviewer added FAQ items
- Check completion screen for FAQ section
